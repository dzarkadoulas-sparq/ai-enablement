# Standup format templates

## Default: Yesterday / Today / Blockers

- **Yesterday** — past tense, outcomes (from commits, PRs merged, or tickets done).
- **Today** — intent (what you plan to do or move).
- **Blockers** — people, systems, or decisions you need, or `None`.

```text
**Yesterday**
- …
- (about five bullets max)

**Today**
- …

**Blockers**
- None
- …
```

## Optional: Asks

If the team uses it:

```text
**Asks**
- …
```

## No coding yesterday

Be honest: PTO, meetings only, or support—then set **Today** to the real plan.

## Async Slack

One message with **bold** section headers. Use bullet characters; only `@mention` if the user approves.

## Brevity

Aim for ~30–60 seconds read-aloud. Offer a longer version only if the user asks (e.g. 1:1 with a lead).

## When data is partial

Add a one-line **Note:** e.g. `*Work items: not connected to Jira MCP—paste your board if needed.*`
